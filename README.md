# 📺 Amazon Prime TV Shows & Movies - Exploratory Data Analysis (EDA)

## 📌 Project Overview

This project performs **Exploratory Data Analysis (EDA)** on the Amazon Prime TV Shows & Movies dataset to uncover meaningful insights about the platform's content library.

The analysis focuses on understanding content distribution, release trends, ratings, genres, countries, and other important characteristics using Python data analysis and visualization libraries.

---

## 🎯 Objectives

- Understand the structure of the dataset
- Perform data cleaning and preprocessing
- Handle missing values
- Explore content distribution
- Analyze Movies vs TV Shows
- Study release year trends
- Analyze content ratings
- Explore country-wise content distribution
- Identify popular genres
- Generate meaningful visualizations and insights
